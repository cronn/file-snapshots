# File Snapshots Documentation

Documentation site for the [file-snapshots](https://github.com/cronn/file-snapshots) monorepo, built with [VitePress](https://vitepress.dev/).

Published at [cronn.github.io/file-snapshots](https://cronn.github.io/file-snapshots/).

## Common Tasks

| Command                   | Description              |
| ------------------------- | ------------------------ |
| `pnpm turbo docs:dev`     | Start development server |
| `pnpm turbo docs:build`   | Build for production     |
| `pnpm turbo docs:preview` | Preview production build |
