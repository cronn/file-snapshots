Add support for elements with `role="insertion"`, `role="deletion"`, `role="mark"`, `role="subscript"` and `role="superscript"`

Snapshot text-level elements with these roles, including their implicit HTML equivalents: `<ins>` (`insertion`), `<del>` (`deletion`), `<mark>` (`mark`), `<sub>` (`subscript`) and `<sup>` (`superscript`).
